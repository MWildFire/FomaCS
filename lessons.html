Подробные уроки

13 отдельных страниц на русском языке — по одной на каждую IB-подтему Theme A. Каждый урок: краткая теория → подробные карточки с реальными примерами → раскрывающиеся экзамен-вопросы с подсказками к ответам. Цель: пройти урок и потом со 100% уверенностью отвечать на экзамене.

Источник: Oxford IB Diploma Programme Computer Science 2025 (Course Companion) + программа NLCS Dubai IB CS. Контент адаптирован на понятный русский язык; английские термины — в скобках. Каждая страница ссылается на соответствующие интерактивные симуляции из основных разделов сайта.

A1 · Основы информатики

A1.1

Аппаратное обеспечение и работа компьютера

CPU, GPU, регистры, шины, fetch-decode-execute, иерархия памяти, SSD/HDD, сжатие, облачные сервисы (SaaS/PaaS/IaaS).

 A1.2

Представление данных и логика

Двоичная/hex, two's complement, ASCII/Unicode, RGB-пиксели, логические элементы, таблицы истинности, K-карты, булева алгебра.

 A1.3

Операционные и управляющие системы

Функции ОС, планировщики, polling vs interrupts, многозадачность (HL), компоненты управляющих систем (HL), реальные приложения (HL).

 A1.4 · HL

Трансляция (HL)

Компилятор vs интерпретатор vs JIT, стадии компиляции, байт-код, линкеры и загрузчики. Сравнительная таблица + use-case grid.

A2 · Сети

A2.1

Основы сетей

LAN/WAN/PAN/VPN, интернет, облако, edge/fog, мобильные сети, протоколы TCP/UDP/HTTP/DNS/DHCP/ARP, TCP/IP слои (HL).

 A2.1.3

Сетевые устройства

Модем, коммутатор, точка доступа, маршрутизатор, межсетевой экран, шлюз — с привязкой к слоям TCP/IP.

 A2.2

Архитектура сетей

Топологии (звезда/шина/кольцо/mesh/tree/hybrid), функции серверов (HL), client-server vs P2P, сегментация (VLAN, subnetting).

 A2.3

Передача данных

Среды передачи, IPv4/IPv6, packet vs circuit switching, маршрутизация, bandwidth/latency, error detection.

 A2.4

Сетевая безопасность

Угрозы (malware, phishing, DDoS, MITM, SQL injection), защита, шифрование (symmetric/asymmetric), сертификаты, MFA, GDPR.

A3 · Базы данных

A3.1

Основы баз данных

DBMS, типы БД, таблицы/ключи (PK, FK, candidate, composite), целостность данных, роли DBA / designer / end user.

 A3.2

Проектирование БД

ER-диаграммы, разрешение M:N через junction-table, нормализация UNF → 1NF → 2NF → 3NF, аномалии, индексы.

 A3.3

SQL и программирование БД

DDL/DML/DCL/TCL, SELECT/WHERE/JOIN/GROUP BY, аггрегации, подзапросы, транзакции и ACID.

A4 · Машинное обучение

A4.1

Основы машинного обучения

AI vs ML vs DL, supervised vs unsupervised, classification vs regression, reinforcement learning, ASIC vs FPGA, этика. ✅ Официальные вопросы.

 A4.2 · HL

Предобработка данных (HL)

Чистка, выбросы (IQR, Z-score), нормализация vs стандартизация, кодирование категориальных, train/val/test split, K-fold CV, утечка данных.

 A4.3 · HL

Подходы к ML (HL)

Linear regression, classification, hyperparameter tuning, clustering, association rules, ANN, CNN, reinforcement learning, генетические алгоритмы.

 A4.4

Этика ИИ

Алгоритмическая предвзятость, surveillance + facial recognition, NLP для cyberbullying, false positive vs false negative — с реальными сценариями.

Как пользоваться

  1. Открой урок по любой подтеме.
  2. Прочитай теорию сверху вниз. Английские термины в скобках помогут на экзамене.
  3. Каждый раз, когда увидишь ссылку на симуляцию — переходи и попробуй! Это самое быстрое запоминание.
  4. В разделе «Тренировочные вопросы» нажми на каждый вопрос, попробуй ответить сам, потом раскрой подсказку.
  5. В конце урока — раздел «Что важно запомнить» = чеклист для экзамена.
  6. Когда чувствуешь уверенность — иди на страницу квиза и проверь себя на 77 экзамен-вопросах с автопроверкой.
