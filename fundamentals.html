Computer fundamentals Основы информатики

Hardware and software, the CPU and memory hierarchy, the fetch-decode-execute cycle, binary / hexadecimal / two's complement, logic gates, operating systems and (HL) the stages of translation from source code to machine code. Железо и ПО, ЦП и иерархия памяти, цикл «выборка-декодирование-исполнение», двоичная / шестнадцатеричная / дополнительный код, логические элементы, операционные системы и (HL) стадии трансляции из исходного кода в машинный.

A1.1

Memory hierarchy — smaller-faster-pricier on top Иерархия памяти — наверху меньше, быстрее, дороже

Click any tier to see typical size, access latency and cost-per-GB. The pyramid shape mirrors the trade-off: capacity grows downward, speed and cost grow upward. Кликните по любому уровню — увидите типовой размер, задержку и стоимость за ГБ. Форма пирамиды отражает компромисс: ёмкость растёт вниз, скорость и стоимость — вверх.

A1.4 · HL

Compiler vs Interpreter — same program, two journeys Компилятор vs интерпретатор — одна программа, два пути

A 3-line program is processed two different ways. Press Step to advance. A compiler does all the translation up-front then runs the result; an interpreter translates and runs one line at a time. Программа из 3 строк обрабатывается двумя способами. Жмите «Шаг», чтобы продвинуться. Компилятор переводит всё заранее, потом запускает результат; интерпретатор переводит и исполняет по строке за раз.

RLE compression Сжатие RLE

A1.2.4 · HL

Karnaugh map simplifier (HL) Упрощение карт Карно (HL)

Click cells (or rows in the truth table) to set 1s. The K-map groups adjacent 1s into prime implicants and reads out a minimised SOP expression. Кликайте по клеткам (или строкам в таблице истинности), чтобы поставить 1. K-map группирует соседние единицы в простые импликанты и выводит минимизированное SOP-выражение.

A1.3.4

Polling vs Interrupts Опрос vs Прерывания

Two CPUs watch the same device. The polling CPU checks on a timer (wasting cycles when nothing happens); the IRQ CPU sleeps until the device signals. Compare CPU load and latency. Два ЦП наблюдают за одним устройством. Polling-ЦП опрашивает по таймеру (тратит такты впустую); IRQ-ЦП спит, пока устройство не сигнализирует. Сравните загрузку и задержку.

A1.3.6 · HL

Thermostat: open vs closed loop (HL) Термостат: разомкнутый vs замкнутый контур (HL)

Open-loop heating runs blindly on a schedule. Closed-loop heating reads a temperature sensor and feeds back: it stabilises near the target even when the ambient changes or a window opens. Разомкнутый контур греет «вслепую» по расписанию. Замкнутый — читает датчик и использует обратную связь: стабилизируется у цели даже при изменении внешней температуры или открытом окне.
